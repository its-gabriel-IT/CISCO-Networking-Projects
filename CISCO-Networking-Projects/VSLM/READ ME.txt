Cisco VLSM Demonstration 
-----------------------------------------------------------------------------------------------------------------------------

Overview

This project puts variable length subnet masking into practice on cisco packet tracer virtual lab. I demonstrate Local Area Networks (LANs) communicate with each other using different classless IP addresses and variable length subnet mask. I also implement basic static routing between routers 
-------------------------------------------------------------------------------------------------------------------------------

Objectives

* Assign different VLSM for each Local Area Network (LANs)
* Implement static routing for each end device
* Implement static routing 
* Demonstrate connectivity between LANs
--------------------------------------------------------------------------------------------------------------------------------

Technologies & Concepts Used

* IPv4 Classless Addressing & Subnetting ( i.e /26/27/28/29/30)
* Static Routing
* Cisco Router Configuration (CLI)
* Interface Configuration (GigabitEthernet, Serial, 802.11)
* Basic Switching (Layer 2 connectivity)
* End-to-End Connectivity Testing (ICMP / ping)

--------------------------------------------------------------------------------------------------------------------------------

Network Architecture

Topology Summary:
* 2 Routers 
* 4 Managed Switches (LAN access layer)
* Multiple end devices (PCs & Laptops)
* IEE 802.33 UTP Ethernet Cable, Serial DCE

--------------------------------------------------------------------------------------------------------------------------------

Skills Demonstrated

* Subnetting across different Local Area Network (LANs)
* Router interface configuration using Cisco Command Line Interface (CLI)
* Layered network troubleshooting methodology

Files Included

* Packet Tracer file (.pkt)
* Network topology diagram
