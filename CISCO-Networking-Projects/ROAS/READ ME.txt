Inter-VLAN Connectivity Router-on-a-stick (ROAS) lab
-----------------------------------------------------------------------------------------------------------------------------
Overview

This project simulates a small business/enterprise environment network built in Cisco Packet Tracer. It demonstrates core networking, segmentation, and security concepts expected for a Junior Network Security Role

-----------------------------------------------------------------------------------------------------------------------------

Objectives

*Design a functional mid-sized enterprise network
*Implement VLAN segmentation
*Enable Inter-VLAN routing (Router-on-a-stick)
*Secure device access using SSH (Disabling cleartext Telnet)
*Provide wired connectivity among several networking devices

------------------------------------------------------------------------------------------------------------------------------

Technologies & Concepts Used 

*VLANs (Virtual LANs)
*802.1q Trunking
*Router-on-a-Stick (ROAS)
*SSH Remote Access
*Switch Port Configuration (Access & Trunk)
*Basic Network Security Hardening
Network Architecture
-------------------------------------------------------------------------------------------------------------------------------

Topology Summary:
* 1 Routers (Cisco R2011)
* 2 Managed Switches (Cisco S2960)
* Multiple end devices (PCs & Laptops)
* Ethernet RJ45 Cables
-------------------------------------------------------------------------------------------

VLAN Addressing Scheme

VLAN 10 (HR):

Network: `192.168.10.0/24`
Gateway: `192.168.10.1`

VLAN 20 (IT):

Network: `192.168.20.0/24`
Gateway: `192.168.20.1`

VLAN 30 (Marketing):

Network: `192.168.30.0/24`
Gateway: `192.168.30.1`

VLAN 40 (Sales):

Network: `192.168.40.0/24`
Gateway: `192.168.40.1`
