Simple LAN Routing Network lab
-----------------------------------------------------------------------------------------------------------------------------
Overview

This project simulates a small local area network environment built in Cisco Packet Tracer. It demonstrates fundamental networking concepts. 

-----------------------------------------------------------------------------------------------------------------------------

Objectives

*Establish LAN Routing 
*Establish LAN Switching
*Implement static IP Addressing

------------------------------------------------------------------------------------------------------------------------------

Technologies & Concepts Used 

*Router Configuration
*Static TCP/IP setting assignment
*Implementing Switching in LAN
*Implementing IEEE 802.3 Ethernet Standards
-------------------------------------------------------------------------------------------------------------------------------

Topology Summary:
* 1 Router
* 1 Managed Switch
* Multiple end devices (PCs & Laptops)
* Ethernet RJ45 Cables
-------------------------------------------------------------------------------------------------------------------------------

Skills Demonstrated

* Implementing Basic Networking concepts on a Local Area Network

-------------------------------------------------------------------------------------------------------------------------------

Files Included

* Packet Tracer file (.pkt)
* Network topology diagram
