# Networking Labs

This repository contains practical networking labs completed using Cisco Packet Tracer to develop hands-on networking skills.

Labs Included

- SOHO Network
- DNS Configuration
- DHCP
- VLANs
- Static & OSPF Routing
- Subnetting 

Each folder contains:

- Project documentation 
- Network topology
- Cisco Packet Tracer (.pkt) file
