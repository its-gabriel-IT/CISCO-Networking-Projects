OSPF CISCO Routing
-----------------------------------------------------------------------------------------------------------------------------

Overview

This project simulates a wide area network (WAN) connection between three geographically separated Local Area Networks (LANs) using Cisco Packet Tracer.
The lab demonstrates how independent networks communicate across routers using Open Shortest Path First (OSPF) routing.

-------------------------------------------------------------------------------------------------------------------------------

Objectives

* Configure and verify WAN serial links between routers
* Assign IP addressing using efficient subnetting (/30)
* End devices should be able to communication between routers
* Implement and validate OSPF routing

--------------------------------------------------------------------------------------------------------------------------------

Technologies & Concepts Used

* IPv4 Addressing & Subnetting
* Open Shortest Path First (OSPF) Routing
* Cisco Router Configuration (CLI)
* Interface Configuration (GigabitEthernet, Serial, 802.11)
* Basic Switching (Layer 2 connectivity)
* End-to-End Connectivity Testing (ICMP / ping)

--------------------------------------------------------------------------------------------------------------------------------

Network Architecture

Topology Summary:
* 3 Routers 
* 2 Managed Switches (LAN access layer)
* 1 Wireless Access Point
* Multiple end devices (PCs & Laptops)
* IEE 802.33 UTP Ethernet Cable, Serial DCE

--------------------------------------------------------------------------------------------------------------------------------

Skills Demonstrated

* OSP Routing between Local Area Network (LANs)
* Efficient subnetting with /30 networks
* Router interface configuration using Cisco Command Line Interface (CLI)
* Layered network troubleshooting methodology

-------------------------------------------------------------------------------------------------------------------------------

Files Included

* Packet Tracer file (.pkt)
* Network topology diagram
